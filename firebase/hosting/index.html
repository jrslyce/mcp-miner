Safety first

No private Codex content is needed to play.

MCP Miner turns abstract work events into game math. The plugin does not need OpenAI credentials, prompts, source code, file paths, repo names, terminal output, browser content, or transcripts.

How to play

Mine while you work, then spend what you earn.

01

Install in Codex

Clone the public repo, run the installer, restart Codex, and trust the MCP Miner hooks.

02

Use Codex normally

Supported work events become abstract score receipts. Those receipts mine Chonks and materials.

03

Fulfill orders

Use mined materials to complete buyer requests, earn Space Bucks, and unlock upgrades.

04

Sync if you want

Link a portal account to see your cloud profile, devices, history, cosmetics, and raw sync receipts.

What data is used

Small receipts in, game progress out.

Used for gameplay

  • Abstract event type and category
  • Server-calculated score and checksum
  • Sequence numbers for conflict detection
  • Aggregated progress, inventory, orders, and upgrades

Not collected for gameplay

  • Prompts, chat transcripts, or source code
  • File paths, repo names, or terminal output
  • Browser contents or OpenAI account credentials
  • Raw private work artifacts

Install

Add MCP Miner to another Codex instance.

Run the installer from a local clone. It backs up your Codex config, repairs standalone MCP-server-only installs, registers the Diamond MCP marketplace, and enables the plugin.

git clone https://github.com/jrslyce/mcp-miner.git cd mcp-miner ruby scripts/install_codex_plugin.rb Windows: powershell -ExecutionPolicy Bypass -File .\scripts\install_codex_plugin.ps1

Portal account

Connect Codex with a short-lived approval code.

1. Start link in Codex

Ask MCP Miner to sync account. Codex creates a temporary link code.

2. Sign in here

Use Google or email/password, then approve the device shown in the portal.

3. Complete in Codex

Codex stores a revocable device token locally and can sync abstract game progress.
